ArrayList class

  • Built-in class that handles “resizeable arrays”
  • Can use with any type!

Type Parameters

  • Inside angle brackets: the type paramter
  • Must be the name of a class
    • Not a primitive!
    • ArrayList<int> doesn’t work!
  • For primitives: use wrapper classes
    • Integer for int
    • Double for double
    • Character for char

ArrayList Methods

ArrayList<Integer> list = new ArrayList<>();
list.add(1); // list[0] = 1
list.add(5); // list[1] = 5
list.add(10); // list[2] = 10
System.out.println(list.get(1)); // should output 5
System.out.println(list.size()); // should output 3
list.set(2, 27); // list[2] = 27
// list.set(25, 3); throws an exception! list size is only 3!

ArrayList Methods

  • add: add to the “end” of this list
    • add(index, element): add element to list at position \(index\), shift everything over
  • get: get the element at index \(i\)
  • set: change the value of the list at position \(i\)
    • Only works if enough things have been added!
  • remove
  • size
