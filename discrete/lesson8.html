Problem Set 2

Uncountability

  • “Infinite binary sequence”? Ex: \((0, 1, 1, 0, 1, 0, \ldots)\)
  • A sequence: for each \(n\), there is a 0 or 1 at position \(n\).
  • Usually defined as a function \(s : \mathbb{N} \to \{ 0, 1 \}\), so \(s(0)\) is the first bit, \(s(1)\) is the second, etc

How would we show that the set of all binary sequences is uncountable? Diagonalization!

Uncountability

Let \(f : \mathbb{N} \to B\) be a function. We will show it’s not a surjection:

Example:

  • \(f(0) = (\mathbf{0}, 0, 0, 0, \ldots)\)
  • \(f(1) = (0, \mathbf{1}, 0, 1, \ldots)\)
  • \(f(2) = (0, 1, \mathbf{1}, 1, \ldots)\)
  • \(f(3) = (0, 1, 1, \mathbf{0}, \ldots)\)

Can we find an infinite sequence \(s\) which is different from each of these?

Different from \(f(0)\) at bit 0, from \(f(1)\) at bit 1, from \(f(2)\) at bit 2, etc?

  • \(s : \mathbb{N} \to \{ 0, 1 \}\) with the property that:
  • \(s(0)\) is different from \(f(0)\)’s bit 0:
    • if \(f(0)\)’s bit 0 = 0, then \(s(0) = 1\)
    • otherwise, \(s(0) = 0\)
  • \(s(n)\) is different from \(f(n)\)’s bit \(n\):
    • if \(f(n)\)’s bit \(n\) = 0, then \(s(n) = 1\).
    • otherwise?

Symmetry

Suppose \(f : A \to B\) is a bijection. Can you find a bijection \(g : B \to A\)?

Inverse function!

  • Prove that \(f\) has an inverse: that is, prove that there is a function \(g : B \to A\) such that \((g \circ f)(x) = x\)
  • Prove that that \(g\) is a bijection.

Transitivity

Suppose \(f : A \to B\) is a bijection and \(g : B \to C\) is a bijection. Can you find a bijection \(h : A \to C\)?

Composition of functions!

  • Define \((g \circ f) : A \to C\).
  • Show that \((g \circ f)\) is a bijection.

Can you find an explicit bijection?

\(f : \mathbb{N} \to \mathbb{N} \setminus \{ 0 \}\) given by \(f(n) = n + 1\) is a bijection.

  • Show that it’s one to one
  • Show that it’s onto

\(\mathbb{Z} \to \mathbb{Q}\)

We have \(f : \mathbb{N} \setminus \{ 0 \} \to \mathbb{Q}\):

\(\mathbb{N} \setminus \{ 0 \}\) \(\mathbb{Q}\)
1 1
2 \(\frac{1}{2}\)
3 2
4 3

Add in: 0, negative integers on left; 0, negative rationals on the right.

\(h : \mathbb{Z} \to \mathbb{Q}\) defined by

\[h(n) = \begin{cases} f(n) &\mbox{if } n > 0 \\ 0 &\mbox{if } n = 0 \\ -f(-n) &\mbox{if } n < 0 \end{cases}\]

\(\mathbb{Q}\) is countable

Since

  • there is a bijection \(f : \mathbb{N} \to \mathbb{Z}\), and
  • there is a bijection \(g : \mathbb{Z} \to \mathbb{Q}\),

By transitivity, \((g \circ f) : \mathbb{N} \to \mathbb{Q}\) is a bijection. So \(\mathbb{Q}\) is countable.

No bijection \(\mathbb{Q} \to \mathbb{R}\)

Proof by contradiction.

  • Suppose \(g : \mathbb{Q} \to \mathbb{R}\) is a bijection.
  • We know there is \(f : \mathbb{N} \to \mathbb{Q}\) that’s a bijection.

By transitivity, there is a bijection \(h : \mathbb{N} \to \mathbb{R}\). This contradicts the proof that no function from \(\mathbb{N} \to \mathbb{R}\) is a bijection.

Predicate Logic

  • Quantifiers: \(\forall\) and \(\exists\)
  • Predicates: \(P(x, y, z, \ldots)\) represents some assertion about \(x, y, z, \ldots\).
    • Mathematical assertions like “\(x + y = z\)” or “\(x\) is prime” or \(x < y\).
    • English assertions like “Person x loves person y at time z”

  • \(\forall x\): “For every \(x\) in our universe…”
    • Prove \(\forall x P(x)\): Let \(x\) be an arbitrary member of the universe. Then (argue why \(P(x)\) must be true).
    • Disprove \(\forall x P(x)\): find one counterexample: one \(x\) in the unvierse with the property that \(P(x)\) is false.

  • \(\exists x\): “There exists (at least one) \(x\) in the universe…”
    • Prove \(\exists x P(x)\): find a specific example of \(x\) in the universe where \(P(x)\) is true.
    • Disprove: Let \(x\) be an arbitrary member of the unvierse. Show that \(P(x)\) is false.

Injections / Surjections / Bijections

Write the statement that \(f : A \to B\) is one to one:

\(\forall x \in A \forall y \in A (x \neq y \rightarrow f(x) \neq f(y))\)

Show that \(f : \mathbb{N} \setminus \{ 0 \} \to \mathbb{N}\) given by \(f(n) = \dfrac{n^2 - n}{2}\) is one to one. Proof idea?

Let \(x, y \in \mathbb{N} \setminus \{ 0 \}\). Prove the contrapositive about \(x\) and \(y\):

\[\mbox{If } \dfrac{x^2 - x}{2} = \dfrac{y^2-y}{2} \mbox{, then } x = y\]

How? Algebra.

Show that \(g : \mathbb{N} \setminus \{ 0 \} \times \mathbb{N} \setminus \{ 0 \} \to \mathbb{N}\) given by \(g(a, b) = a^b - b^a\) is not one to one.

Notice \(g(1, 1) = 1 - 1 = 0\), and \(g(2, 2) = 4 - 4 = 0\).

Write the statement that \(f : A \to B\) is onto.

\(\forall y \in B \exists x \in A (f(x) = y)\)

Show that \(g : \mathbb{N} \setminus \{ 0 \} \times \mathbb{N} \setminus \{ 0 \} \to \mathbb{N}\) given by \(g(a, b) = a^b - b^a\) is onto.

Let \(y \in \mathbb{N}\). Find \(a, b\) such that \(a^b - b^a = y\). Easiest way: choose \(b = 1\)!

\((y+1)^1 - 1^{y+1} = y\).

Again notice: we only need one example for each \(y\). There may be plenty.

Show that \(f : \mathbb{N} \setminus \{ 0 \} \to \mathbb{N}\) given by \(f(n) = \dfrac{n^2 - n}{2}\) is not onto.

Compute some values: \(f(1) = 0, f(2) = 1, f(3) = 3, f(4) = 6\), etc. Seems like \(y = 2\) is missed?

Is there a solution to: \(\dfrac{n^2 - n}{2} = 2\)?

If \(\dfrac{n^2 -n}{2} = 2\), then \(n^2 - n = 4\). Solving this using algebra and the quadratic formula gives us \[n = \dfrac{1 \pm \sqrt{17}}{2}\] which is not a natural number.

Write the statement that \(f : A \to B\) is a bijection.

\(\forall y \in B [\exists x \in A (f(x) = y) \wedge \forall z ( f(z) = y \rightarrow x = z)]\)

“Every \(y \in B\) has one and only one \(x \in A\) mapping to it.”

Or: write down both statements that f is one to one, and that f is onto, and put an \(\wedge\) symbol between them.

Show that \(f : \mathbb{Q} \to \mathbb{Q}\) given by \(f(x) = 2x - 1\) is a bijection.

  • Let \(y \in \mathbb{Q}\). Then \(x = \dfrac{y + 1}{2} \in \mathbb{Q}\) is such that \(f(x) = y\). (onto)
  • If \(\dfrac{x+1}{2} = \dfrac{y+1}{2}\), then \(x = y\). (one to one)
