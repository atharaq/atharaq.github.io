Exam 1

Continuity at a point

\(f(x)\) is continuous at \(x = a\) if:

\[\lim_{x \rightarrow a^-} f(x) = \lim_{x \rightarrow a^+} f(x) = f(a)\]

Example

\[f(x) = \begin{cases} 2x &\mbox{if } x \leq 4 \\ x^2 - 2x + 7 &\mbox{if } x > 4\end{cases}\]

As \(x \rightarrow -2\), eventually \(x < 4\) and so \(f(x) = 2x\). Since \(2x\) is continous (everywhere), \(f(x)\) is continuous at \(x = -2\).

Differentiability at a point

\(f(x)\) is differentiable at \(x = a\) if:

\[\lim_{x \rightarrow a^-} \frac{f(x) - f(a)}{x - a} = \lim_{x \rightarrow a^+} \frac{f(x) - f(a)}{x - a}\]

\[f(x) = \begin{cases}0 &\mbox{if } x < 0 \\ x^2 &\mbox{if } 0 \leq x \leq 1 \\ x &\mbox{if } 1 < x < 2 \\ 2x - 1 &\mbox{if } x \geq 2\end{cases}\]

x = 0

\[\lim_{x \rightarrow 0^-} \frac{f(x) - f(0)}{x - 0} = \lim_{x \rightarrow 0^-}\frac{0 - 0}{x} = 0\]

\[\lim_{x\rightarrow 0^+} \frac{f(x) - f(0)}{x - 0} = \lim_{x\rightarrow 0^+} \frac{x^2 - 0}{x - 0} = \lim_{x \rightarrow 0^+} x = 0\]

The two sides are the same, so \(f(x)\) is differentiable at \(x = 0\).

Recap:

Recall: the quotient rule

\[ (\frac{f(x)}{g(x)})^\prime = \frac{f^\prime(x)g(x) - g^\prime(x)f(x)}{(g(x))^2}\]

and the trig derivatives:

  • \((\sin(x))^\prime = \cos(x)\)
  • \((\cos(x))^\prime = -\sin(x)\)

Trig

\(f(x) = \cos(x)\). \(f^\prime(0) = 0\)? Why?


Derivative of \(\tan(x)\)

Using \(\tan(x) = \dfrac{\sin(x)}{\cos(x)}\) and the quotient rule:

\[ \dfrac{\cos(x)\cos(x) - (-\sin(x))\sin(x)}{(\cos(x))^2} \]

The numerator is \((\cos(x))^2 + (\sin(x))^2 = 1\). So:

\[ (\tan(x))^\prime = \frac{1}{(\cos(x))^2} = (\sec(x))^2 \]

Derivative of \(\sec(x)\)

\(\sec(x) = \dfrac{1}{\cos(x)}\). So again use the quotient rule:

\[ \dfrac{0 - (-\sin(x))}{(\cos(x))^2} \]

We can write this as \(\dfrac{1}{\cos(x)} \dfrac{\sin(x)}{\cos(x)} = \sec(x)\tan(x)\).

Chain Rule

Recall function composition:

  • \((f \circ g)(x)\) means \(f(g(x))\)

Theorem: The Chain Rule. Suppose \(f\) and \(g\) are differentiable. Then \((f \circ g)^\prime(x) = f^\prime(g(x)) g^\prime(x)\).

  • “Derivative of the outside function, plugging in the inside, times the derivative of what’s inside”

Example

Let \(f(x) = \sin(x^2)\). Then \(f^\prime(x)\) = ?

  • Derivative of \(\sin(x) = \cos(x)\)
  • Plug in \(x^2\): \(\cos(x^2)\)
  • Multiply by the derivative of \(x^2\): \(2x\)

So: \((\sin(x^2))^\prime = 2x \cos(x^2)\)
