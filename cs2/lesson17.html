Project 3

Liar’s Dice is a game played with two players (technically it can be played with any number, but let’s stick with two players). Each player has five dice. (This is a stripped down version of the game, so you may see other variants online.) You cannot see your opponent’s dice!

Gameplay

  1. Player 1 “bids”: they state a face value (1-6) and a guess for the minimum number of dice that they believe show that face value.
  2. Player 2 either raises, or calls "LIAR! (challenges). If they raise, they guess a higher number (for the same face value).
  3. Play continues until one player challenges. If a player challenges, the previous player (“bidder”) wins if the number of dice that shows the face value in question is at least as high as the current bid. If not, the “challenger” wins.

Example

  • Player 1’s dice: [1, 2, 1, 4, 6]
  • Player 2’s dice: ???
  • Player 1 bids: face value 1, minimum = 2.
  • Player 2 raises: minimum = 4.
  • Player 1 calls LIAR
  • Player 1 reveals dice: [1, 2, 1, 4, 6]
  • Player 2 reveals: [6, 1, 5, 1, 1]
  • Player 2 wins! There are 5 ones on the table!
