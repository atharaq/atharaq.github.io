Memory Diagrams

Make a memory diagram if \(n = 2\):

public static int f(int n) {
    int a = 0;
    int b = 1;
    if (n <= 0) {
        return 0;
    }

    for (int i = 1; i <= n; i++) {
        int tmp = a + b;
        a = b;
        b = tmp;
    }
    return a;
}

Find the mistakes

Problem: given a String name, return the number of times the character c appears in it. What are all the mistakes in the following?

public static int numAppearances(String name, char c) {
    String name = "bob";
    char c = "b";
    int n = 0;
    for (int i = 0; i < name.length(); i++) {
        if (name.charAt(i) == c) {
            System.out.println(c);
            n++;
        }
    }
    System.out.println(n);
}

Object Oriented Design

  • Classes are templates for data types
  • Represent data and behavior
  • Break your problem up into components: the nouns of your problem.
  • Implement each component in its own class
  • Think about the data required to represnt that component
  • Think about the actions the object should be able to do.

Exercise Part 2

Create a method add in the GaussianInteger class. We add two Gaussian integers as follows: \((a + bi) + (c + di) = (a + c) + (b + d)i\). Test it out in the main class:

GaussianInteger g1 = new GaussianInteger(2, 3);
GaussianInteger g2 = new GaussianInteger(3, -5);
System.out.println(g1);
System.out.println(g2);
g1.add(g2); // should be 5 + -2i
System.out.println(g1);

Exercise Part 3

Create a method multiply in the GaussianInteger class. We multiply Gaussian integers as follows: \((a + bi)(c + di) = (ac - bd) + (ad + bc)i\). Test it out in the main class:

GaussianInteger g1 = new GaussianInteger(0, 1);
GaussianInteger g2 = new GaussianInteger(0, 1);
System.out.println(g1);
System.out.println(g2);
g1.multiply(g2); // should be -1 + 0i
System.out.println(g1);
