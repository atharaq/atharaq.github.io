Loops

Use “length” property to tell the size of the array:

int[] numbers;
// some code that initializes numbers
...

// add up all the elements of numbers
int total = 0;
for (int i = 0; i < numbers.length; i++) {
   total += numbers[i];
}
System.out.println("The total is " + total);

Foreach loop

String[] names;
// some code which initializes names

for (String name : names) {
  System.out.println(name);
}

If we just need the objects in the arrays, and not the index, this is helpful.

Technically called “enhanced for loop.” (Usually just referred to as “foreach”)

Arrays of arrays

We can create arrays of any type we like, using this [] syntax:

int[] x = new int[5]; // creates an array of integers
String[] s = new String[10]; // creates an array of strings

Question: What about arrays of arrays?

Multidimensional arrays

Yes, it works!

int[][] x = new int[5][];
for (int i = 0; i < 5; i++) {
  x[i] = new int[3]; // each element of x is also an array.
  for (int j = 0; j < 3; j++) {
    x[i][j] = i*j;
  }
}

Take a minute to draw a memory diagram for this.

Alternative Syntax

Scanner sc = new Scanner(System.in);
System.out.println("Input a number: ");
int n = sc.nextInt();
boolean[][] sieve = new boolean[n][n];
for (int i = 0; i < sieve.length; i++) {
  for (int j = 0; j < sieve[i].length; j++) {
    x[i][j] = true;
  }
}

Strings

Like arrays, Strings are stored, technically, as references. (Every “object” in Java is stored as a reference to some data).

Immutability

Technically: Strings are immutable. Once a String is created, it cannot be changed. You can, however, change that reference!

String text = "Hello";
String other = text;
String newText = " there";
text += newText;
System.out.println(other);
System.out.println(text);

What is going on here? Draw the memory diagram to explain what is happening.

String API

API: Application Programming Interface.

  • Describes how you might interact with a class / package.
  • In particular, the methods that are available to use.
  • We can access the API for the String class by searching for Java String API.

Some String methods:

  • length(): returns the length of the String
  • substring(): "hello".substring(2) returns “llo”, "okay".substring(1, 3) returns “ka”.
  • equals() and equalsIgnoreCase(): Checks if two Strings are the same (depending on whether you want case sensitivity or not).
  • compareTo(): Compares Strings based on alphabetical order.
  • charAt(): Determines the character at a given position in the String

Example 1

  if (str.equals(s)) {

What does this output if \(list\) is [“a”, “aa”, “ab”, “aa”] and \(s\) is “aa”? What if \(s\) is “ee”? Show your memory diagrams.
