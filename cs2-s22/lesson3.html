Conditionals

If the user enters in a number of seconds below 3600, we output “0 hours”. Perhaps we do not want to output any hours in that case:

String message = originalSeconds + " = " + numHours + " hours, " + numMinutes + " minutes and " + numSeconds + " seconds";
if (hours == 0) {
    message = originalSeconds + " = " + numMinutes + " minutes and " + numSeconds  + " seconds";
}
System.out.println(message);
  • condition must be in parentheses.
  • optional else block.

Scanner API

  • nextInt, nextDouble, nextLong
  • next: reads the next “token” (until whitespace, by default), store as a String
  • nextLine: reads the entire next line (until a newline), store as a String
  • hasNextInt, hasNextDouble, hasNextLine: check if there is more input

hasNextInt may be useful for us in combination with if and while statements

Scanner Pitfall

Scanner sc = new Scanner(System.in);
System.out.println("Enter your age: ");
int age = sc.nextInt();
System.out.println("Enter your name: ");
String name = sc.nextLine();
System.out.println("Hi " + name + ", you are " + age + " years old!");

What happens?

Explanation

  • Scanner object parse input as tokens, not as entire lines.
  • nextInt: parses all of the input up until the end of the integer.
  • “enter” key (newline) not processed by nextInt
  • newline processed by nextLine

Fix? Call nextLine after every group of nextInts.

Fix

Scanner sc = new Scanner(System.in);
System.out.println("Enter your age: ");
int age = sc.nextInt();
sc.nextLine();
System.out.println("Enter your name: ");
String name = sc.nextLine();
System.out.println("Hi " + name + ", you are " + age + " years old!");

Java Structure

Structure

  • Packages (java.util) group related classes
  • classes (Scanner) contain methods (and more…)
  • methods: named sequences of statements

All code goes inside a method!

If statements

if (condition) {
  statement1;
  statement2;
  ...
} else {
  statement3;
  statement4;
}

You need the parentheses. You can get away without curly braces ({ and }) if there is only one statement after the if or else line. Suggestion: always use curly braces. Also use proper indentation to keep your code readable.

Comparison Operators

if (x < 0) {
    System.out.println("x is negative");
} else if (x > 0) {
    System.out.println("x is positive");
} else {
    System.out.println("x is 0.");
}

if (y != 1) {
    System.out.println("y is not 1");
}

if (z >= x) {
    System.out.println("z is bigger than x");
}

Do not use these to compare Strings! More on comparing Strings later.

Boolean Operators

  • &&: conjunction. Evaluates to true if both conjuncts are true.
  • ||: disjunction. Evaluates to true if at least one disjunct is true.
  • !: negation. Evaluates to true if the original condition is false.
System.out.println("Enter a number from 1 to 10: ");
x = sc.nextInt();
if (x < 1 || x > 10) {
    System.out.println("I said 1 to 10!");
}

Boolean variables and methods

  • Boolean operators are often useful with boolean variables and methods
  • Scanner class has a method hasNextInt:
    • true if the next input token can be interpreted as an integer.
    • false if it cannot be interpreted as an integer (ie, the user typed in “hello” or “3.5”)

Example

Scanner sc = new Scanner(System.in);
System.out.println("Input an integer");
if (!sc.hasNextInt()) {
  System.out.println("I said an integer!");
  System.exit(0);
}
int number = sc.nextInt();
System.out.println("You typed in: " + number);

While loops

while loops repeat all the statements inside the curly braces as long as the condition is still true. What does the following code output (keep track of the variables in a state table) in the case where we input 10, 20, or 100?

Scanner sc = new Scanner(System.in);
int x = sc.nextInt();
int i = 0;
while (x / 2 > 0) {
    i++;
    x /= 2;
}
System.out.println(i);

Infinite Loops

Make sure the code inside the while loop actually updates the condition! If it doesn’t, you might never end up exiting the loop, and the program will eventually crash.

while (x / 2 > 0) {
    i++;
}

For loops

The other major loop structure: for loops.

for (int i = 0; i < 10; i++) {
    System.out.println("i is now: " + i);
}

This prints out the numbers 0 - 9.

A for loop has an initializer, a condition, and an update, all in parentheses. That order is important! Most loops can be written either as a for loop or as a while loop.

What does the following code do?

Scanner sc = new Scanner(System.in);
int input = sc.nextInt();
if (input >= 0 && input <= 10) {
    for (int i = 0; i < input; i++) {
      System.out.println(i * i);
    }
}

What does it output when we input 1000? What if we input 10?
